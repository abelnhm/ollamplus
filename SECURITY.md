# Política de Seguridad / Security Policy

## Versión en Español

### Reportar Vulnerabilidades

Si descubres una vulnerabilidad de seguridad en OllamPlus, por favor envíenos un correo electrónico a través de GitHub Issues.

**Por favor incluye:**
- Descripción de la vulnerabilidad
- Pasos para reproducir el problema
- Versión afectada
- Posibles correcciones (si las conoces)

### Tiempo de Respuesta

- **Crítico**: 24-48 horas
- **Alto**: 1 semana
- **Medio**: 2 semanas
- **Bajo**: 4 semanas

### Actualizaciones de Seguridad

Las correcciones de seguridad se lanzarán tan pronto como sea posible y se documentarán en las notas de la versión.

---

## English Version

### Reporting Vulnerabilities

If you discover a security vulnerability in OllamPlus, please send us an email through GitHub Issues.

**Please include:**
- Description of the vulnerability
- Steps to reproduce the issue
- Affected version
- Possible fixes (if known)

### Response Time

- **Critical**: 24-48 hours
- **High**: 1 week
- **Medium**: 2 weeks
- **Low**: 4 weeks

### Security Updates

Security fixes will be released as soon as possible and documented in the release notes.
